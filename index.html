What it is

  • Predictive analysis and data modeling

  • Learns

    Learns

    Supervised
    All data is labeled, and algorithms learn to predict outcomes.
    Unsupervised
    Data is not labeled and the algorithm learns to structure the data.
    semi-supervised
    Some of the data is labeled, but most is not.
    • Supervised
      • Have input and output variables. ML learning the relationship between these and can be queried for the desired output.
      • We as the teacher know the right answer
      • How likely are the Patriots to win this year.
    • Unsupervised
      • The input is known but the output is not
      • Video recognition. Analysis visual aspects in videos along with title and description text. We can't know for certain that a video Kevin records and uploads is about Drupal, but given enough data the model can gain confidence.
      • Clustering
    • Mixed
      • A mix of both.
      • Image Analysis where only some of the pictures have text or metadata associated with them. Trying to categorize pictures of dogs or cat.

    • The 'brain' of the stack
    • Contains 1 (or more) algorithms to build training models
    • Can be queried to make predictions
    • More data === better
